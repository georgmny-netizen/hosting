      Joint Lab Testing / Round Robin Protocol

      Procedure
      1. Batch sample split into Seller / Buyer / Reference Lab units.
      2. Each unit receives identical Trace ID and tamper seal record.
      3. Seller and buyer test using aligned GDMS / ICP-MS method.
      4. Deviation above tolerance triggers third-party reference lab.
      5. Method correlation report is archived in VDR.
      Dispute prevention rule

      Result deviation > ±15% at ppb level requires method reconciliation, sample handling review and calibration evidence comparison before any quality rejection is accepted.
