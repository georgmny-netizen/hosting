SECURITY · ZERO-TRUST RSI BACKEND POLICY

Regional Synergy Index · Server-Side Only Enforcement

RSI must never be calculated from exposed browser secrets. CRM, VDR and compliance database access is restricted to server-side adapters with audit trace logging.
No Secrets in HTML

CRM/VDR/API keys are forbidden in frontend HTML, JavaScript, localStorage, query strings and public config files.

Server Endpoint Only

The browser may call only /api/regional-synergy-index. All database calls must occur behind the backend service boundary.

KYC / Sanctions Block

If KYC or sanctions status is unresolved, RSI returns blocked and the visual score is suppressed.

Audit Trace Required

Every RSI calculation must emit X-YYK-Audit-Trace, timestamp, source status and compliance decision state.
